Вместе к независимости

Все люди должны иметь возможность жить полноценной и независимой жизнью, поэтому главная задача проекта - предоставить им необходимую поддержку и ресурсы для достижения этой цели.

Сервис, создан студентами

Проект был разработан инициативной группой студентов Московского Политехнического университета, которые стремятся сделать мир более доступным для всех. Молодые специалисты из разных областей объединили свои усилия для создания платформы, которая отвечает актуальным запросам людей с ОВЗ.

Целевая аудитория

Люди с ОВЗ, добровольцы, участвующие в благотворительных проектах, неравнодушные люди, готовые присоединиться к волонтерам, опекуны, нуждающиеся в советах по уходу

Цель проекта:

Создание сайта для взаимодействия волонтеров и лиц с ОВЗ в одном из административных округов г. Москва.

Проект способствует:

Социальной интеграции

Сайт поможет людям с ОВЗ получить доступ к информации и ресурсам, улучшая их качество жизни, способствуя интеграции в общество.

Росту числа волонтеров

Увеличение интереса к волонтерству требует удобного инструмента для поиска возможностей участия в социальных проектах.

Упрощению коммуникации

Специализированный сайт обеспечит прямую связь между волонтерами и нуждающимися, делая помощь более быстрой и эффективной.

Поддержке инклюзивного общества

Проект соответствует современным тенденциям создания общества, где все граждане имеют равные права и возможности.

Использованию технологий

Онлайн-платформа отвечает требованиям цифровизации и эффективно решает задачи взаимодействия между социальными группами.

Обратной связи и оценке потребностей

Платформа будет источником информации о потребностях людей с ОВЗ, что поможет организовать необходимую помощь.

Инновация проекта

Сервис предлагает уникальную online-карту, которая помогает людям с ограниченными возможностями здоровья (ОВЗ) оставлять заявки на получение услуг в их городе, а волонтёрам быстро и удобно отслеживать нуждающихся поблизости.

Итоговый результат это:

Бесплатно

сервис предлагает оперативную помощь людям с ОВЗ, что особенно важно в ситуациях, когда требуется срочная поддержка.

Быстро

доступ к качественной поддержке не должен зависеть от материального положения, и делаем всё, чтобы обеспечить равные возможности.

Качественно

работаем только с опытными специалистами, которые обладают необходимыми навыками для оказания помощи людям с ОВЗ.

Надёжно

команда работает 24/7 и состоит из множества волонтёров, готовых оказать пользователям помощь в любой момент.

Журнал проекта

Начнем сначала:

Многие участники проекта были лично знакомы с волонтерской деятельностью, и поэтому с самого старта проекта сайт обрастал всё большим функционалом, позволяющим улучшить жизнь людям с ОВЗ. Всё началось с разработки макета сайта. Нашей главной целью было создать продукт, который станет удобным для каждого. Наши дизайнеры приложили огромные усилия, чтобы сделать интерфейс интуитивно понятным и функциональным. Были созданы такие вкладки как:

  • - О нас
  • - Волонтерам
  • - Почему мы?
  • - Новости
  • - Контакты
  • - Личный кабинет
  • - Карта
Начало разработки:

Далее начался этап написания кода. Разработка разделена на три направления: фронтенд (внешняя разработка), бэкенд (внутренняя разработка) и девопс (разработчик, системный администратор, инженер). На данном этапе задача разработки состоит в доработке дизайна сайта, адаптации его к мобильным устройствам и разработке базы данных, а также создание безопасного хранения данных пользователей. Используемые языки программирования:

  • - Для бэкенда: MySQL, PHP
  • - Для фронтенда: HTML 5, JavaScript, CSS 3
Продвижение:

Во время роста проекта не менее важной частью столо его продвижение. Наша команда начала активно общаться с нашим партером для возможности выхода на новый масштаб проекта. Также мы начали вести социальные сети на популярных площадках: Telegram и ВКонтакте. Мы стараемся как можно чаше выкладывать посты о новых достижениях платформы и познавательные материалы по другим волонтерским организациям. Данная практика позволит быстрее охватить большую аудиторию и привлечь новых инвесторов, что даст скачёк в развитии проекта.
